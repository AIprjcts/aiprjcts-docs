## AI-Assisted Template Development
ai_assistance:
  document_type: "use_case"
  parent_template: "base_template"

ai_prompting_instructions:  
  prompt_refinement_checklist:
    - verify_actors_and_stakeholders
    - check_preconditions_and_triggers
    - ensure_flow_completeness
    - validate_against_business_rules
    - verify_exception_handling

  base_prompt_structure: |
    Create or enhance a detailed use case for ${PROJECT_NAME}.
    
    Provide comprehensive details addressing:
    1. Use Case Overview and Context
    2. Actor and Stakeholder Details
    3. Preconditions and Triggers
    4. Main Success Scenario
    5. Alternative Flows
    6. Exception Flows
    7. Post-conditions
    8. Business Rules and Constraints
    
    Ensure the use case:
    - Is actor-focused
    - Has clear success criteria
    - Includes all necessary flows
    - Maintains traceability
    - Is implementation-agnostic
    - Provides actionable steps

  prompt_templates:
    creation: |
      Generate a comprehensive use case specification:
      Use Case Name: ${USE_CASE_NAME}
      Primary Actor: ${PRIMARY_ACTOR}
      Key Considerations:
      - Business Context: ${BUSINESS_CONTEXT}
      - Primary Goals: ${PRIMARY_GOALS}
      - Critical Requirements: ${KEY_REQUIREMENTS}
      - Success Criteria: ${SUCCESS_CRITERIA}
      - Related Use Cases: ${RELATED_USE_CASES}

    interactive_creation: |
      You will act as an expert use case analyst.
      Begin by asking questions to gather necessary information:
      - Focus on actor goals and interactions
      - Validate preconditions and triggers
      - Ensure all flows are covered
      - Verify business rules compliance
      Present the final use case using the template structure

    enhancement: |
      Act as an expert use case analyst to enhance the existing use case ${DOCUMENT_KEY}.

      **Specific Instructions**:
      - **Review** existing flows for completeness
      - **Identify** missing scenarios or exceptions
      - **Validate** actor interactions
      - **Ensure** business rules compliance
      - **Verify** preconditions and postconditions
      - **Document** all changes
