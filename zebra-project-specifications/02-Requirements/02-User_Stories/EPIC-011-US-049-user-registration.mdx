## Story Details
### Business Value
Enable users to register through multiple channels while ensuring security, compliance, and a smooth onboarding experience.

### User Story
As a new user,
I want to register for the platform through my preferred channel (self-registration or invitation),
So that I can securely create my account and begin using the platform.

## Acceptance Criteria
### Scenario 1: Self-Registration
**Given** I am a new user
**When** I choose to register myself
**Then** I should be able to:
- Enter required information
- Verify my email
- Accept terms and conditions
- Set up security credentials
- Complete initial profile setup

### Scenario 2: Invitation-Based Registration
**Given** I received an invitation
**When** I click the invitation link
**Then** I should be able to:
- View pre-filled information
- Complete missing details
- Set up security credentials
- Verify my identity
- Access my account
